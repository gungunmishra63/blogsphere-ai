# 🚀 BlogSphere AI

A modern blogging platform built using HTML, CSS, JavaScript, Bootstrap, and LocalStorage.

## ✨ Features

* 🔐 User Authentication (Register/Login)
* 👤 User Profile Management
* 📝 Create & Publish Blogs
* ❤️ Like Blogs
* 🔍 Search Blogs
* 📊 Dashboard Statistics
* 📱 Responsive Design
* 🌙 Modern Dark UI
* 🚪 Logout Functionality

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* LocalStorage
